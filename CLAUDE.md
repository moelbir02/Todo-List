## Project Overview

A single-file to-do list web app (`todo.html`) with no build tools or dependencies. Open directly in any browser.

## Architecture

Everything lives in `todo.html` — inline CSS, HTML, and JavaScript in one file. Tasks are persisted via `localStorage`. No frameworks, no bundler, no server required.
